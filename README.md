# Mahar Shwe Exchange

Simple landing page for maharshwe.online.

## Features
- THB ↔ MMK rate display
- Announcement section
- Bank information section
- Telegram contact button
- Dark theme based on provided logo colors

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes
- Current rates are hardcoded demo values from the provided sample.
